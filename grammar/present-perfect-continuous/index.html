G42 · B1

Present Perfect Continuous

현재완료진행형

1
기본 개념BASIC CONCEPT
⭐ TODAY'S KEY GRAMMAR
Present Perfect Continuous
과거부터 지금까지 계속되고 있는 동작을 표현

Present Perfect Continuous → 과거부터 지금까지 계속되고 있는 동작을 표현합니다. 주로 지속 시간이나 현재 영향을 강조할 때 사용합니다.

Present Perfect (결과 강조)와 비교: I've worked here for 5 years. (사실) vs I've been working here for 5 years. (지금도 계속 중)

  • I've been studying for 3 hours.3시간째 공부하고 있다. (지속 강조)
  • She has been cooking since 5 PM.5시부터 계속 요리하고 있다. (진행 중)
2
문법 규칙GRAMMAR RULES
긍정문 · POSITIVE
have/has + been + -ing
I have been working.
부정문 · NEGATIVE
have/has + not + been + -ing
I haven't been sleeping.
FOR + 기간 · DURATION
for 2 hours, for a long time
for 3 days, for weeks
SINCE + 시점 · STARTING POINT
since 8 AM, since Monday
since last year, since this morning
  • have/has + been + 동사-ing 구조로 사용
  • 시간 표현: for 2 hours / since 8 AM / all day / recently
  • 현재 영향: I'm tired because I've been studying
  • 진행 강조 시 현재완료진행 / 완료 강조 시 현재완료 사용
  • 기타 표현: lately, recently, all day, all morning, these days
